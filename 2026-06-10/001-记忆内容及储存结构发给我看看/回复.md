### 二、Session Memories（81 个）

格式：`session-{8位hex}.md`，由 mine session pipeline 的 `digest_to_memory.py` 生成。

**文件结构**（以 `session-cab164bb.md` 为例）：

```yaml
---
name: session-cab164bb
description: <LLM 生成的单句中文摘要>
domain: mined
scope: global
confidence: 0.95          # 由 LLM 评分/会话质量决定
last_accessed: 2026-06-10
hit_count: 0
created: 2026-06-10
---

## Session cab164bb
**Date:** 2026-05-31 12:31
**Summary:** <同 description>
**Tools:** AskUserQuestion, Bash, Edit, Glob, Grep, Read, WebSearch, Write
**Metrics:** 25 turns, 142 tool calls, score=9
```

关键字段：
- **name**: 文件名（不含 .md）
- **description**: LLM 用中文生成的一句话任务总结
- **domain**: 统一为 `mined`，与手工创建的 project memory 区分
- **scope**: `global`（跨项目通用会话）
- **confidence**: 0.55-0.95，反映会话质量和 LLM 摘要置信度
