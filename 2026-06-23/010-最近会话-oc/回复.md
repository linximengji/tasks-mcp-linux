Claude Code 的会话管理分两方面：

**内置命令（有限）**
- `/session list` — 列出最近会话
- `/session switch <id>` — 切换到指定会话
- 没有原生 `/session delete` / `/session clean` 命令

**主动管理方式**

会话文件存储在 `~/.claude/sessions/`，就是普通的 `.json` 文件：

```
~/.claude/sessions/
├── 30856.json  ← 刚用过的
└── 9628.json   ← 旧的
```

你可以直接：
- **删旧会话**: `rm ~/.claude/sessions/9628.json`（文件名就是 session ID）
- **清空所有**: `rm ~/.claude/sessions/*.json`
- **仅保留最近 N 个**: 手动删掉不想要的文件即可

删完重启 Claude Code，`/session list` 就只显示剩下的了。
