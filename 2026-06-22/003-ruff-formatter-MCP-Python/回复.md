**ruff-formatter** 是一个 MCP 服务，把 [Ruff](https://astral.sh/ruff) 包装成 Claude 可直接调用的工具。Ruff 本身是 Astral 公司用 Rust 写的 Python lint + formatter，对标 Flake8 + Black + isort，速度极快（比传统 Python 工具快 10-100 倍）。

你的 `.mcp.json` 里没有配它，它不是当前已注册的 MCP。

---

**对你有没有价值：** 有价值，但不是急迫的。

你大量的 Python 工作（ops-daemon、daily-report、model-proxy、hooks 等）如果能自动 lint + 格式化，可以减少低级问题。但你目前的 CLAUDE.md 里已经写了一套 Python 风格软约束，实际代码质量基本靠人工把控，没暴露出被 lint 问题坑过的情况。

如果配的话，典型用法是让我在写完 Python 代码后自动跑 ruff format + ruff check --fix，省去手动调整缩进、import 顺序、未使用变量这些琐碎事。但需要先确认你的接受度——Ruff 的默认格式化风格（Black 风格）比较激进，比如强制 `"` 还是 `'`、换行策略等，可能和你现有的代码风格有冲突。需要先设好 `pyproject.toml` 里的规则。

**一句话：** 想加的话我可以装，但不加也不影响当前工作流。有想法了说就行。
