飞书语音识别 API：
```
POST /open-apis/speech_to_text/v1/speech/recognize
{
  "speech": { "file_key": "xxx" },
  "config": { "file_type": "opus", "language": "zh" }
}
```

最省事的做法是用飞书自带的 ASR（不需要额外 API key，不需要额外费用）。缺点是飞书 ASR 中文识别准确率一般（95% 左右，受口音/专业术语影响）。
