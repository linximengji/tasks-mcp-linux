你走 proxy + 多模型（deepseek-v4-pro / flash / qwen3.7-max），价格差异大。这个 hook 在每次 Bash 调用后，从 env 或 proxy 取 token 用量，累记到每日文件。不做拦截不弹警告，纯记录。

适合你：关心 token 花费（提过 cost 统计事故），需要月度回顾。

适合你：经常并行多个 agent（看你的 `/loop`, `/deep-research`, workflow），预算控制有需求。
