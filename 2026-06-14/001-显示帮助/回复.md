## Claude Code 帮助

### 常用命令
| 命令 | 说明 |
|------|------|
| `/help` | 显示此帮助 |
| `/clear` | 清空对话上下文 |
| `/compact` | 压缩对话上下文（释放 token） |
| `/config` | 打开配置面板 |
| `/cost` | 查看当前会话 token 用量和费用 |
| `/doctor` | 诊断环境问题 |
| `/init` | 为当前项目生成 CLAUDE.md |
| `/review` | 审查 PR 或 diff |
| `/run` | 运行应用 |
| `/status` | 显示仓库状态 |
